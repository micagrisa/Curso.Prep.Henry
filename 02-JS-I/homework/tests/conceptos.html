Variables

Una variable es un espacio que nosotros reservamos para asignarle un valor el cual puede cambiar. Es como si en una casa reservara una habitacion para guardar cosas, a ese espacio le pones un nombre, por ejemplo, números, y ahi puedo guardar los numeros que quiera, aunque, no solo le puedo asignar numeros, sino lo que yo quiera.... el nombre no limita lo que puedo guardar, sino que sirve unicamente como para yo tener una idea de que es lo que puede llegar a haber dentro de esa habitacion.

Instrumentos de cuerda

Un instrumento de cuerda, es un cuerpo el cual tiene en alguna parte del mismo una o varias cuerdas que pueden ser de diferentes materiales y sirven para hacer sonidos, las cuerdas pueden llegar a ser frotadas (ejemplo el violin), golpeadas (como puede ser un piano) o tocadas (como es el caso de la guitarra).

Funciones

Una función en JavaScript es un procedimiento que le estamos a hacer, pero para que una funcion sea llamada como tal debe tener una variable o argumento y una ejecucion Los argumentos es lo que le da un inicio a la funcion, es lo que le da la orden de que hacer, por decirlo de alguna manera. Mientras que la ejecucion le dice de que forma lo va a llevar a cabo. Por ejemplo la funcion return le da un fin a la ejecución de la funcion y le da un valor para ser devuelto a quien llame a la función.

Declaraciones

Declaracion es una instruccion que damos para que se ejecute algo puntual dependiendo lo que el usuario escriba. por ejemplo If del ingles "SI" se utilisa para decirle al servidor que SI PASA TAL COSA.... VAS A RESPONDER CON TAL COSA.

booleano (true, false)

un booleano es una variable que tiene como respuesta solo dos opciones true o false. Estas respuestas se ejecutan cuando hacemos una declaracion la cual solo puede ser o verdadera (true) o falsa (false)
